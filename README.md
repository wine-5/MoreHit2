# 2025年 12月開催 UnityRoom GameJam

## お題
「もうひとつ」

## ゲーム概要
敵を吹っ飛ばしてさらにヒットを狙おう！

敵に「ストック」を与えよう！　ストックが溜まったら...

### 操作方法

- **移動**：A, D
- **ジャンプ**：スペースキー
- **近距離攻撃**：左クリック
- **射撃攻撃**：右クリック
- **溜め射撃**：Wで溜めて右クリック
